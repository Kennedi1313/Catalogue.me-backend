# Funcionalidades

## User

- Rota para criar usuario
- Rota para editar perfil de usuario
- Rota para fazer login

## Shop

- Rota para criar shop
- Rota para editar perfil do shop

## Items

- Rota para criar item de um shop
- Rota para listar os itens de um shop
    - filtrar por nome, categoria e preço
- Pode ser do tipo product ou work
